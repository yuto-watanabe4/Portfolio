## 苦労したこと
Docker でコンテナ化し、Render へデプロイする際に、ローカル環境と本番環境の差異によるDB接続エラーの解消に苦労しました。

ログを確認しながら原因を切り分け、環境変数や接続設定を一つずつ見直すことで解決しました。
この経験を通して、実装だけでなくインフラやデプロイ周りの重要性も学ぶことができました。
